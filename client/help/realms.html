Creating a realm

Creation

Creating a realm is easy: pick Your Realms -> Create New Realm A new, empty realm will be created for you and you'll be warped to it.

It's currently not possible to delete a realm, but you can create as many as you like (for now).

Editing

When you're inside a realm that you own, the Realm Editor window will automatically appear. This shows a list of all the rooms in the realm. The realm name is shown at the top; clicking on the green hand icon will allow you to change this.

You can use the realm editor to create and destroy individual rooms, as well as warping directly to a specific room.

See the help entry on Rooms for more information about editing rooms.

Inviting people in

What's the point of creating a realm of your own if people can't play in it?

The user interface for doing this is currently very crude. Sorry. Go to your realm; make a note of the instance number displayed on the left side of the title bar. Tell your friends this number.

Now your friends can connect to thickishstring and enter this number in the Warp -> Warp To Instance dialogue. They will then warp to the entrance of your realm.
