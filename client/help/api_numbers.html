API reference: numbers

ThickBasic strings are double precision floating point numbers. For example: 

n1 = 7
n2 = 7.1
n3 = 3.4e9

In addition to the standard operators (+, -, *, %, they support the following operations and methods:

  • n.markup

    Returns a markup object containing this string. This is equivalent to: 

    "{s}"

  • n.toString()

    Returns a string representation of this number.

  • n.toInt()

    Truncates this number to an integer.
