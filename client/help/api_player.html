API reference: Player objects

Players are manipulated via Player objects. These can be found using the Player() function; the Player object for the current player is also available as a special variable called player.

Player objects may be embedded in markup, in which case they expand to be the name of the player. They also support the following methods:

  • player.tell(message)

    Tells the specified player something.

    message must be markup.

  • player.moveTo(destination)

    Moves the player to another room in this realm.

    destination must be a Room object, or a string. If the latter, the name is looked up.
