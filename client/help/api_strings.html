API reference: strings

ThickBasic strings are sequences of characters (including Unicode), delimited with single quotes. For example: 

'Hello, world!'

Special characters can be escaped with a backslash. 

'This\'ll work'

They can also span multiple lines. 

'Line 1
Line 2'

Most values can be converted to a string using the toString property: 

s1 = true.toString
s2 = nil.toString
s3 = (42).toString

Strings are immutable. They support the standard set of comparison operators, and can be concatenated with +. In addition they support the following operations and methods:

  • s.length()

    Returns the length of the string (in Unicode code points).

  • s.markup()

    Returns a markup object containing this string. This is equivalent to: 

    "{s}"

  • s.toInt()

    Returns a number containing the integer value of this string. Produces an error if the string is not a valid number.

  • s.toNumber()

    Returns a number containing the value of this string. Produces an error if the string is not a valid number.

  • s.toString()

    Returns the string itself.
