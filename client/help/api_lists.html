API reference: lists

ThickBasic lists are sequences of other values, delimited with square brackets. For example: 

l1 = [1, 2, 3, 4]
l2 = ['one', 'two', 'three', 'four']
l3 = [1.1, 'string', "markup", nil]
l4 = [[], [1], [1, 2], [1, 2, 3]]

Lists are immutable They can be concatenated with +, and compared by identity (but not by value) with == and !=. In addition, they support the following operations and methods:

  • list[n]

    Returns the nth item from this list. The index is truncated to an integer; out of bound accesses cause an error.

  • list.markup()

    Returns the number of items in this list.
