Frequently Asked Questions

Help! I'm lost!

There's at least one special realm you can always warp to: see the Warp menu. If you get stuck somewhere, you can use this to get home again.

Plus, if you warp to a specific instance when you're already in it, you'll be taken back to the entrance of the realm.

Realms? Instances?

A realm is a world written by a specific author.

Each realm may have one or more instances, which are simultaneous copies of the realm running in parallel. This allows multiple people (or groups of people) to explore the same realm without running into each other.

Right now there's no mechanism for creating new instances, so each realm has exactly one instance. If you look at the title bar the game will tell you the ID of the instance you're currently inside. To warp to another realm, you'll need the author to tell you the instance ID of it; you can then use the Warp To Instance item in the Warp menu to go there.

This is horribly crude and unutterably nasty. It'll be made better at some point. Please bear with me.

What happens when I log off?

Nothing much.

When you log back on again, your character will still be in the same room (provided the author hasn't removed the room, kicked you out, rewritten the realm, etc). When logged off, your character isn't visible to other players.

Can I customise my character?

Not yet. Right now all players have is their names. They don't have descriptions and they certainly don't have sex.

This will be fixed at some point.

What happens if I try to log in twice?

You only have one character, so you can only be logged in once. If you try to connect more than once, your old connection will be automatically disconnected.
